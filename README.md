# Tic Tac Toe Game (Final Project)

# Project Description
This is a simple two-player Tic Tac Toe game built using Python including functions, loops, conditon checking and list based data structure.  
Players take turns entering row and column values to place X or O on a 3x3 board.  
The game checks for a winner or a draw after each move.

# How to Run
1. Open terminal
2. Navigate to the project folder: FinalProject
3. Run the python file : Final Project - Tic Tac Toe.py

# How to Play
1. The game is played between 2 players.
2. Player X starts first.
3. Enter row and column (0-2) to place your mark.
4. The board updates after each move.
5. The game ends when : a player wins or all cells are filled.
6. If board fills up, it’s a draw.
