# Tic Tac Toe Game (Final Project)

# Project Description
This is a simple two-player Tic Tac Toe game built using Python.  
Players take turns entering row and column values to place X or O on a 3x3 board.  
The game checks for a winner or a draw after each move.

## How to Run
1. Open terminal
2. Navigate to the project folder: FinalProject
3. Run the program

## How to Play
1. Player X starts first.
2. Enter row and column (0-2) to place your mark.
3. Players alternate turns.
4. First to get 3 in a row wins.
5. If board fills up, it’s a draw.
